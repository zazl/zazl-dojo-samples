Zazl Optimizer Dojo Samples
All the samples use the Zazl AMD loader to load their modules.The Zazl backend component that performs all the dependency analysis and optimization is running on a Node.js based server. The Warmup samples first page load use the dojo loader (all modules are loaded individually. This first page load initiates the optimization on the server component. Subsequent page loads then use the Zazl loader to load optimized streams of javascript

The source code can be found here

  • Calendar - Loads a dijit/Calendar widget and all it dependencies
  • Declarative - Loads a set of dijit widgets that have been declared in markup
  • Lazy Load - Demonstrates how loading on demand works. A button click loads another widget and all its dependencies
  • Multiple Require calls - Demonstrates multiple top level require calls
  • Warmup - Demonstrates using the Warmup function.
  • Warmup Multiple Require calls - Demonstrate warmup with multiple require calls
