Dojo Samples

Zazl is a Dynamic Javascript Optimizer that enables the loading of AMD modules and all their dependencies in a single response stream without depending on static build tools. It also leverages JavaScript compression tools such as Google Closure, Uglify-js and Dojo Shrinksafe.

The samples use the Zazl AMD loader to load their modules.The Zazl backend component that performs all the dependency analysis and optimization is running on a Node.js based server. Take a look at the javascript network traffic in a browsers developers tools to see the optimization in action.

The samples source code can be found here

  • Calendar - Loads a dijit/Calendar widget and all it dependencies
  • Declarative - Loads a set of dijit widgets that have been declared in markup
  • Lazy Load - Demonstrates how loading on demand works. A button click loads another widget and all its dependencies
  • Multiple Require calls - Demonstrates multiple top level require calls
